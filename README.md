# advanced-hospital-fastapi

## Description
Backend API for managing Doctors, Patients, and Appointments using FastAPI.

## Features
- Doctor CRUD + filter + status update  
- Patient CRUD + search  
- Appointment management  
- JWT Authentication  
- SQLite with SQLAlchemy  
- Swagger API testing  

## Setup
- Clone the repository  
- Install dependencies  
- Run the server  

## Run
python -m uvicorn main:app --reload

## API Docs
http://127.0.0.1:8000/docs

## Modules
- Doctors  
- Patients  
- Appointments  
- Authentication
